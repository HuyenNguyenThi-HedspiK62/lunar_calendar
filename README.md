# lunar_calendar
Khi truy cập một trang web hướng dẫn dạy nấu ăn hoặc trang web thiết kế nhà cửa. Chúng ta thường quan tâm đến hôm nay là ngày mấy trong âm lịch để nấu đồ ăn cho hợp lý hoặc tiện cho việc theo dõi ngày, giờ hoàng đạo. 
Chương trình này tạo ra giải pháp giúp giải quyết vấn đề ấy. 
Tuy nhiên vẫn còn hạn chế là chưa đóng gói để có thể nhúng ngay vào 1 trang web nên sẽ update chỉnh sửa sau.
